# 已支持引擎

## 支持

| 引擎家族 | 典型封包 / 脚本 | 状态 |
| --- | --- | --- |
| Kirikiri Z / KAG | `.xp3`, `.ks`, `.tjs` | 支持 |
| LiLiM / Le.Chocolat AOS | `scr.aos` | 支持 |
| Bruns / EENC / EENZ | `ams.cfg`, `scene/*.bso`, `EENC`, `EENZ` | 支持 |
| 未知非 RPG VN | 先生成引擎报告 | 仅初筛 |

## 排除

| 引擎家族 | 原因 |
| --- | --- |
| LiveMaker / LiveNovel | 本仓库刻意不维护该引擎 |
| RPG Maker | 使用独立的 RPG 工作流项目 |
| DRM / 激活 / 反篡改目标 | 不支持 |
