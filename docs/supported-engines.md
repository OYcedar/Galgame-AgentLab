# Supported Engines

## Supported

| Engine family | Archive/script examples | Status |
| --- | --- | --- |
| Kirikiri Z / KAG | `.xp3`, `.ks`, `.tjs` | Supported |
| LiLiM / Le.Chocolat AOS | `scr.aos` | Supported |
| HSP / DPM | `.dpm` | Supported |
| Unknown non-RPG VN | engine report first | Triage only |

## Excluded

| Engine family | Reason |
| --- | --- |
| LiveMaker / LiveNovel | Intentionally out of scope for this repository |
| RPG Maker | Use a dedicated RPG workflow project |
| DRM / activation / anti-tamper targets | Not supported |
