# 未知引擎初筛流程

步骤：

1. 列出顶层文件、exe、dll、封包扩展名和明显脚本文件。
2. 检查文件头和常见引擎特征。
3. 生成 `work/engine_report.md`。
4. 如果引擎已支持，选择对应 Skill 和 Workflow。
5. 如果引擎未支持，先做最小提取实验，不直接修改本体。

本项目不支持：

- LiveMaker / LiveNovel
- RPG Maker
- DRM、激活、授权校验或反篡改绕过

报告至少包含：

- 疑似引擎。
- 置信度。
- 证据文件。
- 推荐工具链。
- 是否需要解包。
- 可能的文本编码。
- 旧补丁是否只能作为参考。
